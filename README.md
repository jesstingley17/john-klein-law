# Jonathan W. Klein Law Firm Website

A luxurious, black and white law firm website designed with a perfect balance of masculine strength and feminine elegance.

## Features

- **Luxurious Design**: Black and white color scheme with elegant typography
- **Balanced Aesthetics**: Combines strong, masculine elements (bold typography, clean lines) with feminine elegance (serif fonts, refined details)
- **Fully Responsive**: Optimized for all devices (desktop, tablet, mobile)
- **Smooth Animations**: Scroll-triggered animations and transitions
- **Professional Sections**:
  - Hero section with key statistics
  - About section highlighting 28 years of experience
  - Practice areas (Criminal Defense, Family Law, Juvenile Law, Personal Injury)
  - Client reviews section
  - Contact form and information

## Design Philosophy

The website uses:
- **Playfair Display** (serif) for elegant, feminine headings
- **Inter** (sans-serif) for strong, masculine body text
- Pure black (#000000) and white (#FFFFFF) with subtle grays
- Clean lines with refined details
- Smooth transitions and hover effects

## Usage

Simply open `index.html` in a web browser. No build process or dependencies required.

The contact form uses a mailto link to send emails directly to jwk283@aol.com.

## Browser Support

Modern browsers (Chrome, Firefox, Safari, Edge) with support for:
- CSS Grid
- Flexbox
- CSS Custom Properties (variables)
- Intersection Observer API
